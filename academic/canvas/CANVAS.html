⚠️ Análise de Impacto de Falhas

DimensãoCenário de falhaImpacto
Operacional Segmentador ML offline >24h Scores desatualizados; atendentes sem leads priorizados; fallback: regras simples do Java (LN2 hardcoded)
Operacional n8n / WhatsApp indisponível Campanhas não disparadas; fila de ações acumula; fallback: email manual via dealer
Financeiro Falso positivo alto (>10%) no churn Custo de campanha sem retorno; desgaste do cliente com msgs desnecessárias; ROI 300:1 cai para 50:1
Financeiro API Ford indisponível (dados) Pipeline ML para; features desatualizadas; impacto em campanhas do dia seguinte
Reputacional Vazamento de dado (VIN + telefone) Multa LGPD + dano de marca Ford; VIN_Hash SHA1 reduz probabilidade a <0,001%; audit_log trilha acesso
Reputacional Mensagem automática inadequada Cliente insatisfeito publica nas redes; templates aprovados pela Meta limitam variações; revisão humana em escalations
