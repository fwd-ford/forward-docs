- **🟢 Oficial** — fonte primária (Ford, FENABRAVE, Sindipeças, Senacon, FIPE)

- **🔵 Análise/blog** — secundária independente
- **🔴 Bloqueada/paga** — fonte canônica que não foi possível acessar
- **[indicativo]** — número de imprensa, não validado oficialmente

## Notas finais

### Para uso acadêmico
- Toda fonte 🟢 (oficial) pode ser citada com confiança em entregáveis avaliativos.
- Fontes 🟡 (imprensa) devem ser usadas como secundárias, com fonte primária ao lado quando possível.
- Fontes 🔵 (análise/blog) servem como reforço de evidência, não como autoridade isolada.
- **Todas as URLs deste documento foram acessadas em 2026-05-11** (data única de coleta da pesquisa). Em entregáveis futuros, validar se ainda estão disponíveis.

### Para reuso técnico
- A coluna `source` nos CSVs (`fipe_values.csv`, `ford_recalls.csv`, etc.) traz a URL específica de cada linha — permite reproduzir qualquer número em segundos.
- Os arquivos `tmp_research/official/02_external_sources/*.md` contêm a versão narrativa de cada pesquisa, com mais detalhe que este índice.

### Para alimentação contínua
Se o projeto evoluir para coletar mais fontes (ex: Pesquisa III, Sprint 2), recomendação é manter este formato (URL, tipo, descrição, pesquisa de origem) e organizar por tema. Esse documento pode virar `forward-docs/references/external_sources.md` no futuro se ganhar mais conteúdo.
